Demo Optimizely Web

This is a static web page to demonstrate the functionality of Optimizely Web.
Some elements below are modified based on an experiment.

Query parameters

By adding a query parameter at the end of the URL, it is possible to force certain settings within Optimizely.

Disable Optimizely Original variant Variant #1 Increase log level

Hold events

By default, this page does run experiments but does not send any events back to Optimizely. However, events are queued in LocalStorage, so once you allow tracking, all events will be send to Optimizely.
