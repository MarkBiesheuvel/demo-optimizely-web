Flash of content

It is recommended to load the Optimizely JavaScript snippet synchronously. This will ensure that experiment variations are applied as the page loads. If the snippet is loaded asynchronously instead, some content might already be seen to the user before it is modified. This leads to a flash or content or flickering. This page does load the snippet asynchronously and therefore has flickering.
